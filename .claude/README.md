# 💎 Full-Side Engineer Portfolio: Jihye "Toss-Style" Lee

Next.js 16 + Styled Component 기반의 **토스(Toss) 인터페이스** 포트폴리오 시스템

## 🎨 Design Concept: "Extreme Simplicity"

- **Typography**: Toss Product Sans / Pretendard (가독성 최우선)
- **Visual**: 여백의 미, 그림자(Shadow)를 활용한 입체감, 토스 특유의 부드러운 애니메이션
- **UX Flow**: 나열형 경력이 아닌 **'마타수학 리뉴얼'**을 필두로 한 핵심 성과 타임라인

## 🛠 Tech Stack (High Performance)

- **Framework**: Next.js 16 (App Router), React 19

- **Styling**: Framer Motion, Styled-Components

- **State/Data**: React-Query, Recoil, Redux (배민상회 어드민 데이터 패칭 최적화 로직 적용)

- **Efficiency**: AI 자동화 워크플로우 (Claude Code, Gemini, GCP Vertex AI 기반)

## 📂 Project Structure (Toss Service Architecture)

```
src/
├── app/
│   ├── (home)/            # 토스 홈 UI: 마타수학 리뉴얼 등 주요 성과 지표 요약
│   ├── projects/          # [보완] 마타수학 집중 탐구: 전략 제안부터 프론트엔드 구현까지의 Full-Side 기록
│   ├── efficiency/        # '15명 분의 일을 3명이 해결한' 리소스 최적화 및 AI 자동화 기술 상세
│   └── resume/            # 20년 경력의 정수만 담은 텍스트 기반 이력서
├── components/            # 토스 디자인 시스템(TDS) 스타일의 컴포넌트
├── actions/               # Server Actions
└── data/                  # resume.json (마타수학 프로젝트 핵심 데이터 중심)

```

## 📈 Impact-Driven Content (Focus: Mata-Edu)

억지스러운 마케팅 용어 대신, **기술적 성취가 가져온 실질적 서비스 변화**를 기술합니다.

- **[Case 1] AI 마타수학 서비스 리뉴얼 주도 (2023 ~ 현재 · 마타에듀)**
- _Impact_: 서비스 전략 제안 및 2026 리뉴얼 구조 설계를 주도. EBS 초등온에 AI 마타수학을 납품(2025)하며 공교육 플랫폼으로 서비스 영역을 확장했습니다.
- _Technical_: React / Next.js / TypeScript / Redux 기반 개발. AWS CodeCommit 협업 및 Notion QA 관리로 개발 프로세스를 체계화했습니다.

- **[Case 2] EdTech MVP를 빠르게 시장에 (쿨한자 · 패들리 · 2025)**
- _Impact_: 쿨한자(한자 학습)와 패들리(수학 모둠활동) 두 서비스를 기획부터 런칭까지 직접 주도. 아이디어 단계에서 시장 검증 가능한 수준으로 빠르게 출시했습니다.
- _Technical_: 쿨한자는 Next.js 풀스택(Front+Back-end)으로 Vercel 배포. 패들리는 Figma 인터랙션 설계와 Make(자동화)를 활용해 기획 프로세스를 효율화했습니다.

- **[Case 3] AI · 자동화로 개발 생산성 극대화**
- _Impact_: Claude Code, Gemini, GCP Vertex AI를 워크플로우에 도입해 개발 속도와 코드 품질을 동시에 향상. AI 디자인 시스템 자동화로 상점 아이템 제작 파이프라인을 구축했습니다.
- _Technical_: Python 기반 GCP Vertex AI 디자인 생성 자동화 로직 개발. React 기반 프론트엔드까지 전 과정을 직접 주도했습니다.

- **[Case 4] 대형 서비스 어드민 개발 (배민상회 · 우아한형제들)**
- _Impact_: 배민상회 Admin 시스템 및 권한관리 서비스의 데이터 패칭 성능을 최적화하고, Storybook 기반 UI 컴포넌트 시스템을 구축해 팀 개발 생산성에 기여했습니다.
- _Technical_: React / TypeScript / Styled-Components + React-Query 캐싱 최적화. GitLab 협업 및 Jira 스프린트 운영으로 애자일 프로세스를 실천했습니다.

## 🤖 AI Execution Commands

- `/toss-theme-on`: 토스 특유의 화이트톤 UI와 블루 포인트 컬러를 적용합니다.
- `/mata-focus-mode`: 마타수학 프로젝트를 최상단에 배치하고 상세 기술 스택을 강조합니다.
- `/impact-summary`: 20년의 경력을 '기술 기반의 실행 역량' 중심으로 요약합니다.
- `/resource`: 폴더내에 있는 서비스 대표화면을 적용합니다.
